# Employee Performance Analysis & Insights

## Description
A Python-based analytics project that evaluates employee performance using KPI scores. The system generates insights to support performance assessment and decision-making.

## Features
- Load employee performance data from CSV
- Calculate average performance score
- Categorize employees by performance level
- Generate analytical insights
- Validate logic using test cases

## Technologies Used
- Python
- CSV File Handling
- Basic Statistics

## How to Run
Run the program using:
python performance_analysis.py
